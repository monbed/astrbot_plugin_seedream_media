# 火山豆包 - AstrBot 插件

火山方舟 AI 创作插件，集成 **Seedream 图片生成** 和 **Seedance 视频生成** 能力，一个插件搞定图片 + 视频。

## ✨ 功能一览

| 功能 | 指令 | 说明 |
|------|------|------|
| 文生图 | `豆包画图 <提示词>` | 根据文字描述生成高清图片 |
| 图生图 | `豆包画图 <提示词>` + 图片 | 基于单图或多图参考生成新图片 |
| 引用生图 | 回复图片消息 + `豆包画图 <提示词>` | 使用引用消息中的图片作为参考 |
| 头像参考 | `豆包画图 <提示词>` + @某人 | 使用被 @ 的全部用户头像作为参考 |
| 文生视频 | `豆包视频 <提示词>` | 根据文字描述生成短视频 |
| 图生视频 | `豆包视频 <提示词>` + 图片 | 基于单图或多张参考图片生成动态视频 |
| 引用生视频 | 回复图片消息 + `豆包视频 <提示词>` | 使用引用消息及当前消息包含的多张图片统一生成视频 |
| 头像视频 | `豆包视频 <提示词>` + @某人 | 使用被 @ 的全部用户头像生成动画 |

## 🚀 快速开始

### 1. 安装插件

在 AstrBot 插件市场搜索 `astrbot_plugin_seedream_media` 安装，或手动放入 `plugins` 目录。

### 2. 获取 API Key

前往 [火山方舟控制台](https://console.volcengine.com/ark) 创建 API Key，并开通 Seedream / Seedance 模型的使用权限。

### 3. 配置插件

在 AstrBot 管理面板中填写以下配置：

| 配置项 | 必填 | 说明 | 默认值 |
|--------|------|------|--------|
| `VOLC_API_KEY` | ✅ | 火山方舟 API Key | 空 |
| `model_version` | ✅ | 图片模型 ID，如 `doubao-seedream-5.0-lite-260128` | 空 |
| `video_model_version` | ❌ | 视频模型 ID，如 `doubao-seedance-1-5-pro-251215`，留空则禁用视频功能 | 空 |
| `image_size` | ❌ | 图片分辨率（`1K` / `2K` / `3K` / `4K`） | `2K` |
| `video_resolution` | ❌ | 生成视频的分辨率（`480p` / `720p` / `1080p`），1.0 lite 只持 720p，图生视频不支持 1080p | `720p` |
| `video_ratio` | ❌ | 生成视频的宽高比（`16:9`、`4:3`、`1:1` 等 7 种比例）。`adaptive` 会自动根据画面或提示词选择。 | `adaptive` |
| `video_duration` | ❌ | 生成视频的时长。填 2\~12 的整数表示指定秒数。填 `-1` 表示由模型基于 4\~12 内自动选择。 | `5` |
| `VOLC_ENDPOINT` | ❌ | API 端点地址 | `https://ark.cn-beijing.volces.com/api/v3` |
| `show_prompt_in_reply` | ❌ | 回复中是否显示提示词 | `true` |
| `clean_cron` | ❌ | 每天清空媒体缓存文件夹的定时策略 (Cron格式) | `0 4 * * *` |
| `download_timeout` | ❌ | API调用和媒体下载的全局连接超时时间 (秒) | `120` |

> **提示**：不同模型版本支持的分辨率不同 — Seedream 5.0 lite 支持 2K/3K，Seedream 4.0 支持 1K/2K/4K。

## 📖 使用示例

**文生图**
```
豆包画图 星空下的大海，高清，细节丰富
```

**图生图** — 发送图片的同时输入指令
```
豆包画图 将图片中的人物换成古风造型
```

**引用生图** — 回复他人发的图片消息
```
豆包画图 改成二次元风格
```

**头像参考** — @某人，自动使用其头像
```
豆包画图 换古装，细节丰富 @小王
```

**文生视频**
```
豆包视频 女孩在森林里奔跑，阳光透过树叶
```

**图生视频** — 发送图片的同时输入指令（支持多图组合输入）
```
豆包视频 让画面动起来，镜头缓缓推进
```

**头像参考视频** — @某人，自动使用其头图进行动态生成
```
豆包视频 这个人怎么这么好笑 @老李
```

> 视频生成通常需要 **3~5 分钟**，完成后会自动发送，请耐心等待。

## ❓ 常见问题

**Q: 提示 "API KEY无效或未授权"**
- 检查 `VOLC_API_KEY` 是否正确填写
- 确认 API Key 已开通对应模型的使用权限

**Q: 提示 "视频模型未配置"**
- 在插件配置中填写 `video_model_version`，如 `doubao-seedance-1-5-pro-251215`

**Q: 图生图/图生视频失败**
- 参考图片需为可公开访问的 URL，避免有防盗链的图片
- 图片格式建议 JPG/PNG，大小不超过 5MB

**Q: 提示 "调用频率超限"**
- 插件内置 10 秒防抖，请勿频繁发送
- 火山方舟 API 有频率限制，建议适当间隔

## ⚠️ 注意事项

- API Key 为敏感信息，请勿泄露给第三方
- 图生图 / 图生视频仅支持公开可访问的图片 URL
- 请遵守火山方舟平台使用规范，合理使用 API
